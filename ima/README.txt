========================================
Google Interactive Media Ads SDK for iOS
========================================

This is the Google IMA SDK for iOS.

Requirements:
- Xcode 3.2 or later.
- Runtime of iOS 4.0 or later.

The latest documentation and code samples are available at:
http://code.google.com/apis/ima/docs/sdks/googleios_ads.html


The SDK uses json-framework. More information: http://code.google.com/p/json-framework.
